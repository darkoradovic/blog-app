# Blog App - NextJS + TypeScript

## 📝 Project description

Full stack NextJS 13 blog app build from scratch using Prisma, PostgreSQL, Tailwind CSS, Typescript , Next Auth and a lot more.

## 💻 Project Features

- Firebase Storage
- Next authentication
- Create new blogs

## 🚀 Technologies and tools used

- NextJS
- ReactJS
- TypeScript
- Firebase
- Tailwind CSS
- Prisma
- PostgeSQL
